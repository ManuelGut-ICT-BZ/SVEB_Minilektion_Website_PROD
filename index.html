Hands-on

Hands-on - Vorbereitungsauftrag vor Lektion

  1. Visual Studio Code herunterladen: Link zum Download
  2. Visual Studio Code mit vorausgewählten Standardeinstellungen installieren
  3. Visual Studio öffnen
  4. Oben Links im Menü "File / Auto Save" aktivieren
  5. Auf der Linken Seite auf die Icons mit den 4 Quadraten klicken
  6. Ganz oben "HTML CSS Support" eingeben und auf install klicken
  7. Ganz oben "Live Server" eingeben und auf install klicken
  8. Hands-on Template herunterladen: Link zum Download
  9. Zip-Datei entzippen
  10. Oben Links im Menü "File / Open Folder" auswählen
  11. Den vorhin entzippten Ordner auswählen und öffnen
